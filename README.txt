AI PROCTOR - AI-POWERED EXAM MONITORING FOR MOODLE
===================================================

AI Proctor provides comprehensive exam monitoring and academic integrity tools for Moodle using advanced computer vision and AI technology.

FEATURES
--------
* Real-time face detection and tracking using MediaPipe AI
* Multiple person detection
* Phone and device detection  
* Tab switching and window focus monitoring
* Screenshot capture on violations
* Detailed violation reports with evidence
* Student evidence review system
* Ban management for repeat offenders
* Integration with Moodle quiz module

REQUIREMENTS
------------
* Moodle 4.0 or higher
* Modern web browser with webcam support
* JavaScript enabled

INSTALLATION
------------
1. Extract the contents of this ZIP file
2. Copy the ai_proctor folder to /blocks/ directory
3. Visit Site Administration > Notifications
4. Complete the installation process
5. Configure settings at Site Administration > Plugins > Blocks > AI Proctor

USAGE
-----
1. Add the AI Proctor block to a course
2. Configure monitoring settings
3. Students will be prompted to enable their webcam during quizzes
4. View violation reports in the block
